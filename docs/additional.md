## PA-010: フォーカス投票モード（チャレンジモード）

**概要**: ランキングページのプラグインカード上の「vote」ボタンをクリックした際、そのプラグインを固定して他のプラグインと連続対戦させる機能

**要件**:
- クリックしたプラグインが投票ペアの片方として固定される
- 対戦相手はスマートマッチング（同程度のELO帯）で選択
- 投票後も同じプラグインで連続対戦（相手は変わる）
- 「チャレンジモードを終了」ボタンで通常モードに戻れる

**チェックリスト**:
- [x] チケット作成
- [x] 実装
  - [x] `selectOpponentForPlugin()` 関数追加 (matchmaking.ts)
  - [x] Vote API にフォーカスモード対応追加
  - [x] VoteComparison コンポーネントにフォーカスモードUI追加
  - [x] 翻訳追加（日本語/英語）
- [x] ビルド確認
- [ ] デプロイ

## 運用ガイド

### 新規要件の追加方法

1. このファイルの「要件一覧」テーブルに追加
2. チェックリスト付きの要件詳細セクションを追加
3. `docs/tickets/PA-XXX_*.md` に詳細チケットを作成
4. `docs/tickets/README.md` のチケット一覧を更新

### ステータス定義
- **Ready**: 着手可能
- **In Progress**: 作業中
- **Done**: 完了
